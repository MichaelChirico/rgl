INTRODUCTION

These pages document the Javascript classes that support the rgl package for display in web pages. Most of the methods should be assumed to be internal and subject to change.

The methods are organized by source file:

controls.src.js: methods for controls
draw.src.js: methods for drawing
init.src.js: methods for initialization
mouse.src.js: methods for mouse handling
pieces.src.js: methods for transparency
projection.src.js: methods for projections
rglClass.src.js: obsolete methods
rglTimer.src.js: methods for players
selection.src.js: methods for selection
shaders.src.js: methods for shaders
subscenes.src.js: methods for subscenes
textures.src.js: methods for textures
utils.src.js: utility methods

This file was produced on 2021-05-05 for rgl version 0.106.15.
